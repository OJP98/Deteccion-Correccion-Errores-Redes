IMPORTANTE:
Se implementó el algoritmo de Hamming y CRC, por lo que tanto el cliente como el servidor reciben de parámetro en consola

- hamming
- CRC


PARA USAR EL PROGRAMA:

1. Iniciar el programa "server.py" con línea de comando, incluyendo luego el tipo de algoritmo a utilizar.

	Ejemplo: python3 server.py hamming


2. Iniciar el programa "client.py" con línea de comando, también incluyendo el algoritmo a utilizar

	Ejemplo: python3 client.py hamming


3. Dentro de server.py, ingresar el mensaje a enviar así como la tasa de errores por bit.


NOTA: Se debe usar el mismo algoritmo en ambos programas, de lo contario el programa tendrá errores.
